## Install

Go must be installed. [Link](https://golang.org/)

Add this repository to your environment by executing this command:

```cmd
go get github.com/eliamartani/go.blog
```

## Run

Windows

```cmd
go build

go.blog.exe
```

Others

```cmd
go run *.go
```

## Maybe you will need

MySQL Driver:

```cmd
go get github.com/go-sql-driver/mysql
```
